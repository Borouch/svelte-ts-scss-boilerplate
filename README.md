# esbuild Svelte Typescript Starter 

This is a starter kit to build [Svelte](https://svelte.dev) containing:
- Svelte 3.x
- Typescript && aliases
- [esbuild](https://github.com/evanw/esbuild) bundling

## Get started

Install the dependencies...

```bash
npm i
```

Then, run and serve your app:

```bash
npm run dev
npm run serve
```
